	JoKaShi - Protótipo de software de busca para a organização de arquivos
	
		Há duas formas de executar esse software: 
	
	1 - Compilar da IDE (Eclipse - Java IDE):
			1.1 - Basta abrir a IDE
			1.2 - Importar a pasta do projeto como um Java Project
			1.3 - A partir disso, é só clicar com o botão direito na classe Main que está 
					dentro do package gui e seguir os seguintes passos:
						Run As -> 2 Java Application
		
	2 - Compilar pelo terminal:
			2.1 - Entra a pasta src e compila todos os arquivos .java com o seguinte comando:
					"java *.java -d <nome_do_diretorio_de_destino>"
			2.2 - 	Entra no diretório de destino escolhido e rodar o seguinte comando:
					"java gui.Main"
			2.3 - A partir disso é só interagir com o sistema.
